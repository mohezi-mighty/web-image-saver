Privacy Policy

Web Image Saver
Last updated: 2026-04-05
Web Image Saver fetches and converts the selected image locally inside Chrome so the user can save it in PNG, JPEG, or WEBP format.

1. Overview

Web Image Saver is a Chrome extension that helps users save standard webpage images in a user-selected format. The extension processes images locally in the browser and does not provide online services, user accounts, cloud storage, analytics, or advertising features.

2. Information the Extension Accesses

The extension may access the following information only when the user uses the extension on an image:

  • The original image URL, for the purpose of fetching the selected image data
  • The selected image bytes, for the purpose of local decoding and conversion
  • User preference settings stored locally, such as output format and quality values

3. How Information Is Used

The extension uses local image data only to:

  • Fetch the selected image requested by the user
  • Decode and re-encode the image into PNG, JPEG, or WEBP
  • Generate a filename based on the user's local settings
  • Save the converted image to the user's device

4. Data Sharing and Transmission

The extension does not send image data, filenames, browsing history, or personal information to external servers. The extension does not sell, transfer, or share user data with third parties. The extension does not use analytics, advertising SDKs, or remote logging.

5. Data Storage

Web Image Saver stores only local preference values by using chrome.storage.local. These values include default format, JPEG quality, WEBP quality, whether to ask for a filename before saving, and the filename pattern. The extension does not store image files or user content on external servers.

6. Remote Code

The extension does not use remote code. All executable code required for the extension is packaged with the extension itself.

7. User Control

The extension runs only when the user right-clicks an image and chooses a save command. Users can change local settings from the options page, stop using the extension at any time, or uninstall it through the browser.

8. Contact

If you have questions about this Privacy Policy, please contact the publisher through the contact method provided on the extension listing page.
